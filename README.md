# Rhymedle

A daily poetry word game where you complete the rhymes in famous poems.

## How to Play

1. Fill in the missing rhyming words in an excerpt from a famous poem
2. Choose from multiple choice options (Easy Mode) or type the words yourself (Hard Mode)
3. You have 3 attempts to get all rhymes correct
4. After completing the rhymes, guess the poem's author
5. A new Rhymedle is available every day!

### Setup

```bash
npm install
npm run dev
```

### Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
